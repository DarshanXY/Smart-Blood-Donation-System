# Smart Blood Donation System 🩸

A web-based application to manage blood donors and blood requests efficiently.

## 🚀 Features
- Add donor details
- Search donors by blood group
- Delete donor records
- Data stored using LocalStorage

## 💻 Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Description
The Smart Blood Donation System helps users manage donor information and quickly find suitable donors during emergencies. It provides a simple interface with dynamic data handling and persistent storage.
